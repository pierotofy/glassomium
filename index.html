Create HTML5 multi-touch applications!

By running everything in texture-mapped web browser windows, you can focus on creating applications rather than spending time learning a new complicated framework or API.

In fact many web applications are ready to be ported on Glassomium, with no changes!

Focus on the application, not on the platform!

You can use your favorite language to develop for Glassomium, as long as it runs in a web browser! That includes HTML5, CSS, Javascript, Flash, PHP, Ruby, ASP.NET.
  • Multi-touch gestures for zooming, rotating and dragging windows
  • Adobe Flash support. Think about all those games, YouTube, Hulu...
  • TUIO-to-JS support for developing true multi-touch applications
  • Fullscreen application support
  • Small API. Yes, this is a feature!
  • Customizable user interface
  • Cross-platform (currently Windows and Linux)
  • Apache 2.0 Commercial-friendly license
